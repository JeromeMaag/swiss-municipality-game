# Roadmap
## Gameplay Ideas

- Guided learning mode
- Settlements or place names instead of municipalities
- Historical municipality datasets as a game mode
- Configurable round counts, such as 5, 10, or 20 rounds
- Endless practice mode
- Hint system with score penalties
- Daily challenge
- Challenge links so multiple players can play the same rounds
- Multiplayer options

## Accounts

- Google login
- Email verification
- Password reset by email
- Username changes
- Account deletion
- Guest-to-account migration for games played without an account

## Stats And History

- Best and worst cantons
- Exact-hit rate
- Streaks and achievements
- Leaderboards

## Map And UI

- Better admin geodata screen
- Pin drop animation
- Reveal transition animation
- Score count-up animation
- Keyboard and accessibility polish

## Data And Admin

- Admin dashboard for game statistics
- Geodata update history
- Active dataset selection in the admin
- Better geodata import job status and progress
- Admin action for cleaning up local development data
- Historical municipality imports

## Production Hardening

- Login rate limiting
- Tracking endpoint throttling
- Content Security Policy
- Self-host Leaflet assets
- Non-default admin path
- CodeQL
- Dependency lockfile workflow
- Production deployment documentation

## Sharing

- Shareable result text
- Result cards or screenshots
- Spoiler-safe summary sharing
